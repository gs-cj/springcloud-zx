管理服务解决方案架构师

岗位职责：

1. 海外管理服务市场策划、推广及项目应标；（管理服务Managed Services，OSS、网优及CEM相关服务）；

2. 承担海外管理服务市场市场销售业绩，参与方案规划及定制；

3. 参加各类型专业展会，Workshop等市场活动。


任职要求：

1. 精通管理服务方案架构设计，对市场有敏锐的分析能力，主动积极的进取心态，勇于开拓创新；

2. 具备5年以上管理服务大项目投标支持经验，熟悉OSS/SOC/CEM/WFM等运维工具方案，能独立承担管理服务市场策划及推广、投标项目支持及项目管理；

3. 独立完成管理服务服务产品方案的定制及成本概算，对客户CxO级别人员进行宣讲交流；

4. 具备团队精神、钻研精神，以及不断主动学习的能力，工作责任感强；

5. 英语听说读写流利，丰富的宣讲经验及文案能力，具备通讯技术基础，有专业服务经验者优先。

工作地点：深圳


管理服务总工

岗位职责：

1. 负责SQM/CEM、云计算、SDN/NFV、运营支撑等领域的管理服务战略规划预研，确定长期的业务演进方向和路径；

2. 负责创新服务产品规划、开发的指导，推动服务产品方案落地；

3. 负责管理服务国内外高端论坛、重要客户的主题发言，以及技术交流和需求引导。


任职要求：

1. 本科毕业10年以上、硕士毕业8年以上电信/IT运维专业、SQM/CEM、云计算、SDN/NFV等至少一个或多个领域的工作经验，并有带领团队工作经验； 

2. 对运营商运营运维转型面临的挑战和痛点有深入理解和独到的见解；

3. 具有较强沟通、协调能力、团队合作组织能力，熟练的英文读、写、听、说能力； 

4. 能适应国内外短期出差，全年累计出差约40%。

工作地点：深圳


大数据运维架构师

岗位职责：

1. 负责服务领域大数据应用场景梳理、算法建模、数据挖掘和UX设计呈现；

2. 负责服务领域大数据应用的研发推进、验证闭环和质量保证；

3. 负责建立和培养服务领域大数据应用团队，支撑大数据运维市场；

4. 负责服务领域大数据应用国内外高端论坛、重要客户的主题发言，以及技术交流和需求引导。


任职要求：

1. 本科毕业10年以上、硕士毕业8年以上大数据分析、挖掘、建模经验，有实现客户收益提升的案例；并有带领团队工作经验；

2. 具有运营商前端市场和后端运维的丰富工作经验或者丰富的互联网企业IT系统运维工作经验；

3. 具有较强沟通、协调能力、团队合作组织能力，熟练的英文读、写、听、说能力；

4. 能适应国内外的短期出差，全年累计出差约30%。

工作地点：深圳


外线光缆总监

岗位职责：

1、负责外线光缆业务整体规划、落地推进、监控汇报，确保外线项目顺利执行，梳理资源需求并提前做好规划；

2、参与外线光缆重点项目的投标、交付、运维；

3、指导外线光缆项目工程方案编制、宣讲，开发工程方案模板、宣讲模板等；

4、把关外线设计、监理、施工等外包资源质量，协助外包招标及资源池建设；

5、梳理、维护、更新外线业务流程；

6、负责外线预算工具系统的维护，指导外线项目预算编制；

7、负责外线相关岗位的培训、认证、面试等工作；

8、负责外线业务专业知识技能的推广、专业能力的提升。


任职要求：

1、具备10年以上外线光缆相关工作经验，5年以上海外通信工程项目交付管理经验，其中至少2年以上大型外线项目的项目经理经历；

2、具有很强的客户沟通和管理能力，具有和运营商高层管理人员（CXO级别）交流汇报的经历；

3、优秀的英语能力，具备与国外客户无障碍沟通交流和宣讲的能力；

4、具有很强的责任心、执行力、沟通协调能力，团队管理能力，能承担较大压力；

5、对国际电信工程项目的执行和交付模式非常熟悉，有国际高端运营商或跨国运营商项目交付经验者优先；

6、具备PMP认证资质或通信概预算证书者优先；

7、有设计院工作经验者优先；

8、能适应不定期海外出差。

工作地点：深圳、南京、西安、上海


国际服务交付项目管理经理

岗位职责：

1、负责协助和指导服务合同执行，提供业务支持； 

2、负责指导服务执行团队的组建和日常工作开展； 

3、负责对实施项目的支持、监控、跟踪和管理； 

4、服务执行管理流程、体系的建立和完善。


任职要求：

1、教育程度：大学本科或以上学历； 

2、相关工作经验：三年以上通讯行业工作经验，一年以上项目管理相关经验，一年以上海外项目管理经验者优先； 

3、知识：具有良好的通讯知识，具备电信网络运营或维护经验； 

4、技能：良好的团队沟通能力和协调能力，英语听说读写良好；

5、能适应海外出差（一年中约60%的时间出差海外）。

工作地点：深圳
